About section

The TerraMania project started in 2022, when i (VexagonGames) and my friend decided to make a farming game.
The first version wasn't the best as i was new to coding in C++ and didnt have OpenGL knowledge.
The oldest version of TerraMania is currently lost media but i might find it somewhere.
Terramania's version 1a3 is currently being developed and is the most feature-rich version.

Currently the project members are:
  • Sirb0zo (content design help)
  • Eliseo (composer)
  • VexagonGames (main programmer, artist)
  • JimGames (secondary programmer)
